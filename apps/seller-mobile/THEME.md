# Mech Bazar UI Theme Tokens

This file documents the exact design tokens extracted from the [mechbazar.com](https://mechbazar.com) website to be used for the Vendor App redesign.

## Colors

### Brand Colors
- **Primary Red:** `#db0000` (Used for primary buttons, active states, icons, and error states)
- **Secondary Navy:** `#02427A` (Used for header/nav backgrounds, secondary actions, and contrast elements)
- **Deep Navy:** `#003C70` (Used for borders and deep accents)

### Backgrounds & Surfaces
- **App Background:** `#fafafa` (Light gray for main app background)
- **Surface / Card:** `#ffffff` (White for cards, modals, and sheets)
- **Surface Hover:** `#f5f5f5`

### Text Colors
- **Heading / High Contrast:** `#222222`
- **Body / Medium Contrast:** `#333333`
- **Caption / Low Contrast:** `#777777`
- **Inverse Text (on Primary/Navy):** `#ffffff`

### Feedback / Semantic
- **Success:** `#28a745`
- **Warning:** `#ffc107`
- **Error:** `#db0000` (Same as Primary Red)
- **Border / Divider:** `#e0e0e0`

## Typography
- **Font Family:** System default sans-serif (Inter / Roboto) to keep native feel but match web's clean grotesque look.
- **Weights:** 
  - Regular (400) for body
  - Medium (500) for buttons and tabs
  - Bold (700) for headings

## Components

### Buttons
- **Primary:** Background `#db0000`, Text `#ffffff`, Border Radius `8px`, no shadow.
- **Outline:** Transparent background, Border `1px solid #db0000`, Text `#db0000`.

### Inputs
- **Background:** `#ffffff`
- **Border:** `1px solid #cccccc` (Default), `1px solid #db0000` (Focused)
- **Radius:** `8px`
- **Padding:** `12px 16px`

### Cards
- **Background:** `#ffffff`
- **Border Radius:** `12px`
- **Shadow:** Subtle shadow `0px 2px 8px rgba(0,0,0,0.05)`
- **Padding:** `16px`
